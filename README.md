# Customer-Sentiment-Analysis
🌸 FlowerAura Customer Sentiment Analysis

This project analyzes customer reviews from the FlowerAura website using Python and Natural Language Processing (NLP) techniques. The reviews were collected through web scraping, cleaned and preprocessed using Pandas, and analyzed using TextBlob to determine customer sentiment.

The project also includes visualizations such as Sentiment Distribution, Average Rating vs Sentiment, Word Clouds, and Review Length Analysis to uncover customer opinions and provide business recommendations.

🚀 Project Workflow
Web Scraping using BeautifulSoup
Data Cleaning & Preprocessing with Pandas
Text Preprocessing (Lowercasing, Tokenization, Stopword Removal, Lemmatization)
Sentiment Analysis using TextBlob
Data Visualization using Matplotlib & Seaborn
Business Insights & Recommendations
🛠️ Technologies Used
Python
BeautifulSoup
Pandas
NLTK
TextBlob
Matplotlib
Seaborn
WordCloud
📊 Key Insights
Most customer reviews were positive.
Positive reviews were slightly longer than negative reviews.
Customers appreciated timely delivery and flower quality.
Delivery experience and product quality can be further improved.
